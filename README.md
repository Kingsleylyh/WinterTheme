# Santa's Winter Journey

## Background Story
It is Christmas Eve, and Santa is racing against time to deliver presents across a snow-covered city. Suddenly santa encountered someone that lock him up because she wanted to own all the presents instead of distributing to everyone. Firstly, you will need to guide the raindeer to pick up presents and save santa. After successfully saving santa, you will need to deliver the presents as quick as possible while running away from hostile cars racing over you. Christmas can only be save if you deliver all the presents in the limited time, or eliminating everyone chasing you!

## How to Get the Project (Cloning)
1. Open a terminal.
2. Clone the repository (replace with your repo URL if needed):
   `git clone https://github.com/Kingsleylyh/WinterTheme.git`
3. Enter the project folder:
   `cd WinterTheme`


## Requirements
- Python 3.10+ recommended 
- pygame
- opencv-python 

Install dependencies:
```
pip install pygame opencv-python
```

## Run the Game
From the project folder:
```
python main.py
```

## Controls
### Global / Menus
- ENTER: Continue / Start
- SPACE: Skip story videos
- ESC: Pause

### Level 1 (Gift Run)
- W/A/S/D: Move
- Left Shift: Boost

### Level 2 (City Chase)
- W: Accelerate
- S: Brake / Reverse
- A / D: Steer
- SPACE: Drift
- Left Shift + W: Nitro Boost
- Y: Shoot (auto-aims at nearby enemies)
- ESC: Pause
